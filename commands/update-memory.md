Update the project's CLAUDE.md with learnings from this session.

## Steps

1. **Read the current CLAUDE.md** in the project root

2. **Review the conversation** for:
   - New conventions discovered or established
   - Bugs found and their root causes
   - Architectural decisions made
   - Gotchas encountered
   - Files modified and why
   - TODO items identified
   - Known issues discovered

3. **Update CLAUDE.md** with:
   - Add to "Recent Changes" with today's date
   - Add any new "Known Issues"
   - Update "TODO" list
   - Add any new "Gotchas" discovered
   - Update "Conventions" if new patterns were established

4. **Keep it concise** - CLAUDE.md should be scannable, not verbose

Rules:
- Only add verified information from this session
- Don't remove existing entries unless they're resolved
- Date-stamp recent changes
- If CLAUDE.md doesn't exist, suggest running /init-project first
